# Podniky a zaměstnavatelé Brno–Starý Lískovec
## Podkladový list pro pexeso a kampaňové materiály

Starý Lískovec je primárně obytné sídliště, ale nabízí překvapivě pestrou paletu zaměstnavatelů — od nadnárodních technologických firem přes místní výrobce až po zemědělské družstvo přežívající v katastru panelového sídliště.
